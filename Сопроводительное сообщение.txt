1) для создания базы данных, необходимо:
	- в appsettings.Development.json откорректировать строку подключения
	- указать проект Web как startup project
	- зайти в Package Manager Console(Visual Studio) и выбрать там Default project "src\Infrastructure"
	- ввести в Package Manager Console команду "update-database"
После выполнения этих действий будет создана база данных по пути, указанном в строке.
